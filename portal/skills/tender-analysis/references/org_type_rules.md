# 机构类型识别规则

## 采购单位类型分类体系

### 一级分类

| 类型代码 | 类型名称 | 说明 |
|---------|---------|-----|
| EDU | 教育机构 | 学校、学院、大学、培训机构等 |
| GOV | 政府机关 | 政府部门、行政单位、委办局等 |
| MED | 医疗卫生 | 医院、卫生院、疾控中心等 |
| ENT | 企业单位 | 国企/央企/民企/上市公司等 |
| MIL | 军警部队 | 军队、武警、公安、消防等 |
| RES | 科研院所 | 研究院、研究所、检测中心等 |
| OTH | 其他 | 无法归类的机构 |

---

## 关键词匹配规则（优先级由高到低）

### 教育机构 (EDU)
**必含关键词**：
- 大学、学院、学校、中学、小学、幼儿园、职业技术、高职、高中、初中
- 教育局、教育厅、教委
- 研究生院、附属学校

**排除词**：军事院校归入 MIL 类

### 政府机关 (GOV)
**必含关键词**：
- 政府、市政、区政、县政、乡政、镇政
- 委员会、管理局、管理委员会、办公室、发展和改革
- 人民代表大会、政协、纪委、监察
- 财政局、统计局、审计局、税务局、海关
- 城管、市监、应急管理、生态环境

### 医疗卫生 (MED)
**必含关键词**：
- 医院、卫生院、卫生所、诊所、医疗中心
- 疾病预防控制中心、疾控、卫生局、卫计委
- 血液中心、急救中心、妇幼保健

### 军警部队 (MIL)
**必含关键词**：
- 部队、军区、武警、公安、消防救援、边防
- 海警、警察、监狱、司法厅、检察院、法院
- 军事院校（含"军"字的学校）

### 科研院所 (RES)
**必含关键词**：
- 研究院、研究所、科学院、实验室、工程院
- 检验检测中心、质量检测、计量院
- 科技园、创新中心（需无企业性质词）

### 企业单位 (ENT)
**关键词**：
- 有限公司、股份公司、集团、工厂、厂
- 国有、央企（含上述词即可归类）

---

## 特殊处理规则

1. **多词命中**：按优先级取最高级别（MIL > GOV > EDU > MED > RES > ENT > OTH）
2. **"附属"前缀**：附属医院→MED，附属学校→EDU
3. **"国家"前缀**：国家XX局→GOV，国家XX院→RES
4. **含"集团"但也含"学校"**：优先 EDU（教育集团情况）
