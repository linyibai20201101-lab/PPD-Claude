## 附录 A：业务重合度判断框架

> 用于判断目标公司与我司的竞争/合作/供应关系，可从「产品重合」与「市场重合」两个维度定位。

![业务重合度示意图](assets/f66903d29cf78edb3a6432465870744775d7b5e5c23dc60d3332688e328a69d7.png)

| 象限 | 产品重合 | 市场重合 | 典型关系 |
|------|----------|----------|----------|
| 竞争 | 是 | 否 | 同类产品、不同区域或客户群 |
| 高度竞争 | 是 | 是 | 直接竞品，需重点监控 |
| 潜在竞争 | 否 | 是 | 品类不同但争夺同一预算/渠道 |
| 不构成竞争 | 否 | 否 | 可探索合作或供应链整合 |
